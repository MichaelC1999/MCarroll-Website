MapApp

Repo - Backend Repo

An application for sharing posts about different locations. This application uses React for the frontend, making use of MapBox API for the interactive map. The backend divides the data into two seperate collections; entries and markers. Markers are rendered to the map by their Lat/Long and make a request to the backend when clicked. This request sends the entry ID and fetches the entry data.

Burger Stacker

Repo

This app allows users to create a custom burger and order it. The user must authenticate to order. Once the user is logged in, they can also view prior orders. This application was built entirely in React and is connected to a Firebase database, using the out-of-the-box authentication. The React features used for this project include Redux and Routing.

Social Platform

Repo - Backend Repo
