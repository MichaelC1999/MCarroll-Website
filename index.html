En Eff Tee Marketplace

Repo

MapApp

Repo - Backend Repo

An application for sharing posts about different locations. This application uses React for the frontend, making use of MapBox API for the interactive map. The backend divides the data into two seperate collections; entries and markers. Markers are rendered to the map by their Lat/Long and make a request to the backend when clicked. This request sends the entry ID and fetches the entry data.

Staking dApp

Repo

This application is a minimalist approach to allowing users to lock in ETH to stake and receive interest in an ERC-20 token. I created MichaelCoin to be used as the token for portfolio demonstration. Value locked in can be unstaked at any time. This is a concept dApp that can be used as a transparent way to receive investment funding in return for tokens that will (presumably) have future use or value. This dApp is live on the Ethereum Rinkeby test network.
